1) Fortran (1957)
Интересный факт: Fortran был одним из первых языков высокого уровня и стал основным языком для научных и инженерных вычислений.

2) Lisp (1958)
Интересный факт: Lisp известен своей уникальной синтаксической структурой и считается первым языком, поддерживающим концепции обработки списков и рекурсии.

3) COBOL (1959)
Интересный факт: COBOL (Common Business-Oriented Language) был разработан для бизнес-приложений и до сих пор широко используется в банковских системах.

4) C (1972)
Интересный факт: C был создан для разработки операционной системы Unix и стал основой для многих современных языков, включая C++, C# и Java.

5) Smalltalk (1972)
Интересный факт: Smalltalk был одним из первых языков, поддерживающих объектно-ориентированное программирование, и оказал значительное влияние на развитие интерфейсов.

6) Java (1995)
Интересный факт: Java была создана с идеей «пишешь один раз — запускаешь везде», что сделало его популярным для разработки кросс-платформенных приложений.

7) Python (1991)
Интересный факт: Python стал популярным благодаря своей простоте и читаемости, что делает его идеальным языком для новичков и быстрого прототипирования.

8) JavaScript (1995)
Интересный факт: JavaScript изначально был создан за всего за 10 дней, и его первоначальное название было Mocha; он стал основным языком для веб-разработки.
